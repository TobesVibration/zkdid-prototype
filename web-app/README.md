# Web App (Dapp)

## How to run
- clone project repository
- `cd web-app`
- `npm install`
- `npm run dev`

## User Stories
As a user I would like to
- Access the web app
- Connect my wallet
- View my domains (NFTs held)
- Create a new domain
    - Input value
    - Generate proof
    - Mint NFT
    - Store proof in DWN
- Verify a domain sent to me (stretch)
